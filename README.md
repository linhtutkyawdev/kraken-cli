# Kraken Cli

```sh
cargo install krk
krk create
```

```sh
cargo shuttle run
```

```sh
cargo shuttle deploy
```

## Tech Stack
- Rust
- Shuttle.rs
- Axum
- Askama
- Tailwindcss
- Htmx
